# Remotion 使用示例

## 示例 1：基础用法

```
用户：帮我创建一个 React视频 相关的基础组件

AI 应该：
1. 参考本技能中的最佳实践
2. 生成符合规范的代码
3. 添加必要的类型定义
4. 包含基本的错误处理
```

## 示例 2：性能优化

```
用户：这个组件渲染太慢了，帮我优化一下

AI 应该：
1. 分析当前代码的性能瓶颈
2. 应用本技能中的优化策略
3. 使用缓存或 memoization
4. 减少不必要的重渲染
```

## 示例 3：代码审查

```
用户：帮我审查这段代码，看看有没有问题

AI 应该：
1. 检查代码是否遵循本技能的规范
2. 识别潜在的 bug 和安全隐患
3. 提出具体的改进建议
4. 给出优化后的代码示例
```
